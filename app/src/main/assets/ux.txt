UX / экранные схемы
1. Общая концепция навигации

Платформа: Android
Тип навигации:

Нижняя панель (Bottom Navigation) на 4–5 основных разделов

Плавающая кнопка действия (FAB) для быстрого добавления (посадки, задачи, заметки)

Основные разделы нижнего меню (иконки + подписи):

«Главная» — дашборд «Сегодня на даче»

«Участок» — карта грядок и посадок

«Растения» — каталог культур и сортов

«Задачи» — календарь/список дел и напоминаний

«Ещё» — дневник, магазин, профиль, настройки (через меню)

2. Главный сценарий: вход и первый запуск
Экран 2.1. Onboarding (серия из 3–4 экранов)

Цель: коротко объяснить пользу приложения.

Ключевые экраны/слайды:

«Все посадки под контролем» — картинка грядок, краткий текст.

«Напоминания о поливе и подкормках» — картинка календаря/капелек.

«Подбор сортов под ваш регион» — карта РФ + растения.

«Дневник урожая» — фото ведра с томатами/ягодами.

Контролы:

Листание свайпом.

Кнопки: «Пропустить», «Далее», на последнем — «Начать».

Экран 2.2. Выбор режима / авторизации

Варианты:

Кнопка «Продолжить без регистрации» (гостевой режим, с плашкой «Данные могут потеряться при смене телефона»).

Кнопка «Войти / синхронизировать»:

Google

По номеру телефона (опционально, в будущем).

3. Экран «Главная» (дашборд «Сегодня на даче»)
Экран 3.1. Главная

Цель: собрать в одном месте состояние дачи «на сегодня».

Основные блоки:

Блок погоды:

Текущая температура, иконка (солнечно/дождь).

Краткий прогноз на 3 дня.

Важные предупреждения: «Ночью возможны заморозки до –2 °C».

Блок «Сегодняшние задачи»:

Список 3–5 ближайших задач на сегодня/завтра:

Полить грядку «Томаты теплица».

Подкормить «Клубника ремонтантная».

Кнопка «Посмотреть все» → экран «Задачи».

Блок «Мои грядки/участок»:

Миниатюра схемы участка, 1–2 яркие грядки с «сигналами» (например, красная точка: просрочена задача).

Блок «Последние записи» (дневник):

1–3 карточки: фото + краткая заметка:

«Посадил рассаду помидоров».

«Появилась тля на смородине».

Блок промо/магазин (опционально):

Небольшая карточка «Семена томатов под ваш регион» → раздел магазина.

FAB (плавающая кнопка «+»):

При нажатии — мини-меню быстрых действий:

«Добавить посадку»

«Добавить задачу»

«Добавить запись в дневник»

«Добавить грядку»

4. Раздел «Участок» (карта грядок)
Экран 4.1. Список / карта грядок

Режим представления:

Вкладка «Список» (по умолчанию) — карточки грядок.

Вкладка «Схема» — условная карта/план (позже может быть рисовалка).

Элементы:

Переключатель «Список / Схема» сверху.

В списке: карточки вида:

Название грядки: «Теплица 1», «Грядка №3, морковь».

Подпись: какие культуры (иконки).

Индикаторы:

Наличие задач.

Состояние (например, значок «полив нужен» при просрочке).

Экран 4.2. Карточка грядки

Цели:

Показать, что и где посажено на конкретной грядке.

Дать быстрый доступ к задачам и журналу именно по этой грядке.

Блоки:

Название + иконка грядки.

Список посадок на грядке:

«Томат «Бычье сердце» — 10 кустов (рассада, высажена 12.05)».

Задачи, связанные с грядкой:

Ближайшие 2–3 + кнопка «Все задачи».

Журнал:

Последние записи (фото, заметки).

Кнопки:

«Добавить посадку»

«Добавить задачу»

«Добавить запись»

5. Раздел «Растения» (каталог → сорт → рекомендации)
Экран 5.1. Каталог растений

Поиск + фильтры:

Строка поиска (по названию культуры/сорта).

Фильтр по группам:

Табы/чипы: «Овощи», «Фрукты», «Ягоды», «Цветы», «Деревья», «Кустарники», «Зелень»…

Список:

Карточки культур:

«Томаты»

«Огурцы»

«Яблоня»

«Клубника» и т.д.

Экран 5.2. Список сортов выбранной культуры

Фильтры и «умные» параметры:

«Регион РФ» (подставляется по умолчанию из профиля, можно менять).

Переключатели:

Теплица / Открытый грунт / Оба.

Высота (низкий / средний / высокий).

Кислотность (для тех культур, где актуально).

Список сортов:

Карточки:

Название сорта, краткая характеристика.

Значки: морозостойкость, раннеспелость, урожайность.

Экран 5.3. Карточка сорта (variety)

Блоки:

Название сорта (локализованное).

Фото (если есть).

Краткое описание.

Агротехника:

Сроки посева/высадки.

Схема посадки (расстояние, глубина).

Требования по освещённости, поливу, подкормкам.

Морозостойкость / регионы:

«Рекомендуемые регионы: ЦФО, ПФО, СЗФО»

«Хорошо переносит морозы до –25 °C (условная зона X)».

Умные фильтры (отображение):

Значок теплица/ОГ.

Высота куста, тип роста.

Кнопки:

«Добавить на грядку» (→ выбор грядки → подтверждение).

«Добавить задачу по этому сорту».

«Перейти к покупкам» (→ магазин, если есть привязанные товары).

6. Раздел «Задачи» (календарь и список)
Экран 6.1. Вкладки «Список» / «Календарь»

Вкладка «Список»:

Фильтры: «Все / Сегодня / Неделя / Просроченные».

Карточки задач:

Заголовок: «Полив томатов в теплице».

Дата/время.

Привязка: грядка/культура/сорт.

Чекбокс «выполнено».

Вкладка «Календарь»:

Месячный/недельный календарь с точками/цветами по дням:

Количество задач в день.

При нажатии на день — список задач снизу.

Экран 6.2. Создание/редактирование задачи

Поля:

Название (автогенерация при выборе действия, но можно редактировать).

Тип задачи:

Посев / Высадка / Полив / Подкормка / Обработка / Сбор урожая / Другое.

Привязка:

К грядке, культуре, сорту — через выпадающие списки.

Дата и время (поддержка повторений — «каждую неделю», «каждые 3 дня»).

Напоминание:

За N часов/дней до.

Комментарий.

7. Дневник (журнал дачника)
Экран 7.1. Список записей

Элементы:

Список карточек:

Дата.

Фото (если есть).

Заголовок / первые строки заметки.

Теги: грядка, культура, сорт.

Фильтры:

По периоду (сезон, год).

По грядке/культуре.

Экран 7.2. Просмотр записи

Фото (галерея, свайп).

Дата, привязки.

Текст заметки.

Экран 7.3. Создание записи

Фото (камера/галерея).

Текст.

Привязка к грядке/культуре/сорту.

Оценка состояния (иконки: «всё хорошо», «болезни», «вредители», «урожай»).

8. Погода и предупреждения
Интеграция в «Главную» + отдельный экран

Экран 8.1. Детальная погода

Прогноз на 7–10 дней:

Температура, осадки, ветер.

Специальный блок «Риски для растений»:

«Ночью заморозки — риск для рассады томатов».

«Сильный ветер — лучше подвязать высокие растения».

Настройки (в профиле):

Локация (автоопределение + возможность задать вручную).

Порог заморозков, при котором слать предупреждения.

9. Магазин (маркетплейс / партнёрка)
Экран 9.1. Витрина

Категории: Семена, Удобрения, Средства защиты, Инвентарь.

Рекомендации «под мой участок»:

По регионам/культурам/сортам, которые пользователь ведёт.

Экран 9.2. Карточка товара

Название, цена, фото.

Краткое описание.

Ссылка «Купить на…» (переход во внешний маркетплейс).

Привязка к сортам/культурам (если есть).

10. Профиль, настройки, подписка
Экран 10.1. Профиль пользователя

Имя/ник.

Способ входа (Google/телефон/гость).

Локация/регион.

Настройки единиц измерения (°C, м² и т.п.).

Экран 10.2. Подписка

Статус: «Бесплатно с рекламой» / «Подписка активна до …».

Карточки тарифов:

«Месяц», «Год», их цена.

Список плюсов подписки:

Нет рекламы.

Больше грядок/записей/фото (если планируется).

Расширенная аналитика/статистика.

11. Карта экранов и переходов (упрощённо)

Onboarding → Выбор режима → Главная

Главная:

Погода → Детальная погода

Задачи → Раздел «Задачи»

Мини-карта участка → Раздел «Участок»

Последние записи → Дневник (конкретная запись)

FAB → «Добавить посадку / задачу / запись / грядку»

Участок:

Список/Схема грядок → Карточка грядки → Добавить посадку / задачу / запись

Растения:

Каталог → Список сортов → Карточка сорта → (1) Добавить на грядку, (2) Создать задачу, (3) Перейти в Магазин

Задачи:

Список/Календарь → Карточка задачи → Редактирование

Дневник:

Список → Запись → Редактирование

FAB → Создать запись

Ещё:

Профиль

Подписка

Магазин

Настройки
