Рекомендуемая последовательность

Группа → крупные корзины по назначению (плодовые, овощные, ягодные, декоративные, зелень/пряные).

Культура/вид → конкретный вид (яблоня, смородина, томат…).

Сорт/гибрид → список популярных сортов/гибридов с краткими тегами (ранний/зимостойкий/самоплодный и пр.).

Полезные UX-штрихи: поиск по вводу в каждом списке, быстрые теги-фильтры (зимостойкость, срок созревания, высота/тип куста, самоопыляемость), «избранное».

Как это вывести в интерфейсе

Меню №1 (Группа) — список из 5 пунктов.

Меню №2 (Культура/вид) — динамическое наполнение по выбранной группе.

Меню №3 (Сорт/гибрид) — список популярных сортов + мини-теги:

плодовые/ягодные: зимостойкость, самоплодность, срок (ранний/средний/поздний), высота/тип кроны;

овощи: тип (индет/дет), раннеспелость (дни), теплица/ОГ;

декоративные: зона зимостойкости, высота, освещенность, период цветения.

Опционально сверху ставим быстрые фильтры (по региону/зоне зимостойкости, типу посадки — ОГ/теплица), чтобы список сортов сразу «подчищался» под условия пользователя.

Если хочешь, я превращу это в готовые JSON-справочники (по трём уровням) и добавлю «теги-свойства» для каждого сорта, чтобы их можно было фильтровать и показывать иконками.
